## Overview

Joined an intensive two-week software engineering bootcamp focused on web development fundamentals. I practiced building responsive, user-friendly websites using HTML, CSS, and JavaScript while gaining hands-on experience with Git for version control and modern development workflows. The program gave me a solid understanding of how real-world web applications are structured, built, and deployed — forming the foundation of my journey as a developer.

---

## What I Learned

This bootcamp was crucial because it gave me the fundamentals I needed. I learned how to structure HTML semantically, style websites with CSS, and add interactivity with JavaScript. More importantly, I understood the importance of responsive design and how to use Git for version control — essential skills for any developer working in teams or on production projects.

## Impact & Takeaway

This two-week intensive was a game-changer for me. It transformed me from someone curious about web dev into someone who could actually build things. The bootcamp's emphasis on hands-on projects meant I wasn't just learning concepts — I was building real websites, debugging actual problems, and learning how professionals work. This foundation gave me the confidence to continue learning more advanced technologies like React and Node.js later on.
